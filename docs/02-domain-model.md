# 1. Ubiquitous Language

The following glossary defines the official business terminology adopted throughout the project.

Every concept described here must be consistently used across the documentation, source code, APIs, database model and technical discussions.

Maintaining a common language reduces ambiguity and helps preserve the integrity of the domain model.

---

| Term | Definition | Primary Actors | Common Actions |
|------|------------|----------------|----------------|
| Organization | A company or institution that subscribes to the platform. Each organization owns its users, courses, permissions and business data. | Organization Administrator | Create, Activate, Update, Suspend |
| Organization Administrator | User responsible for managing an organization's workspace, users and learning environment. | Organization Administrator | Invite Users, Manage Departments, Configure Organization |
| Department | Organizational unit that groups employees according to the company's structure. | Organization Administrator, Manager | Create, Update, Archive |
| Team | Operational group of employees working together within or across departments. | Manager | Create, Assign Members, Archive |
| Employee | Platform user who consumes learning content assigned by the organization. | Employee | Enroll, Complete Courses, Take Assessments |
| Instructor | User responsible for creating and maintaining learning content. | Instructor | Create Courses, Publish Lessons, Update Content |
| Manager | User responsible for monitoring employee development and team performance. | Manager | Monitor Progress, View Reports, Assign Learning Paths |
| Course | Structured learning content composed of modules and lessons. | Instructor, Employee | Create, Publish, Enroll, Archive |
| Module | Logical subdivision of a course used to organize lessons. | Instructor | Create, Update, Reorder |
| Lesson | Smallest learning unit within a course. | Instructor, Employee | Publish, View, Complete |
| Learning Path | Ordered collection of courses designed to achieve a specific learning objective. | Manager, Employee | Create, Assign, Complete |
| Assessment | Evaluation used to measure knowledge acquisition during a learning process. | Instructor, Employee | Create, Submit, Grade |
| Certificate | Official document issued after successfully completing a learning path or course. | Employee | Generate, Validate, Revoke |
| Enrollment | Relationship representing an employee's participation in a course or learning path. | Employee | Enroll, Cancel, Complete |
| Notification | Message generated by the platform to communicate relevant events to users. | System | Send, Read, Archive |
